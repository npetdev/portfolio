# 🌐 Personal Portfolio Website

This is my personal portfolio website built with **React**, **TypeScript**, and **Vite**.  
The goal of this project is to present my skills, projects, and experience.

## 🚀 Tech Stack

- **React**
- **TypeScript**
- **Vite**
- **SCSS / Sass**
- **Framer Motion**
- **EmailJS**

## 📂 Project Structure

Folder structure:

src/
├── animations
├── assets/ # images, icons
├── components/ # UI + common + home + project components
├── context/ # theme context provider
├── pages/ # home page 
├── styles/ # global styles, mixins, variables
├── App.tsx
└── main.tsx


## 🎯 Features

- Modern hero section  
- About me section  
- Skills with categories  
- Projects with filters  
- Contact form with email integration  
- Light/Dark theme  
- Smooth animations  

## 🛠️ Setup

Install dependencies:

npm install


Run the development server:

npm run dev
